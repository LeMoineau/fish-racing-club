# fish-racing-club
Fish racing game

## Lancement

- ouvrir `index.html` dans votre navigateur
- appuyer sur la touche `espace` ou cliquer en haut à gauche ou en bas à gauche de la page
- vous êtes maintenant sur le jeu `Fish racing club`
